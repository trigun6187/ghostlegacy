# Cyberpunk Header Aesthetic Update Plan

## Goal
Implement a premium, high-tech gaming aesthetic for 'About' and 'What I Do' section headers by replacing dashes with Cyberpunk Brackets and adding a subtle background glow.

## Context
- Target elements: `h2` elements with class `.section-title` in [`index.html`](index.html).
- Existing line element is generated by `.section-title::after` in [`styles.css`](styles.css).

## Plan
1. **Update `styles.css` for Header Text Styling & Background Glow**: Modify `.section-title` to apply `letter-spacing: 3px;`, `text-transform: uppercase;`, and add the subtle background radial gradient (`background: radial-gradient(ellipse at center, rgba(255,71,87,0.05) 0%, transparent 70%);`).
2. **Update `styles.css` (Line Removal)**: Completely remove the existing definition for `.section-title::after` (the colored line).
3. **Update `styles.css` (Brackets)**: Add new definitions for `.section-title::before` and `.section-title::after` to construct the L-shaped corners using borders, positioning, and the specified gradient (`#ff4757` to `#7c3aed`) with a `box-shadow` glow.
4. **Update `index.html`**: Verify that no explicit HTML elements are creating the dashes.
